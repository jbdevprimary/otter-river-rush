# Product Overview

**Otter River Rush** is a mobile-first 3-lane endless runner game featuring Rusty the Otter navigating a rushing river.

## Core Identity

- **Platform**: Unified Expo app (web + iOS + Android)
- **Genre**: Endless runner with lane-based gameplay
- **Target**: Mobile-first (phones/tablets primary, desktop secondary)
- **Distribution**: Web via GitHub Pages, native via EAS Build
- **License**: MIT (open source)

## Key Features

- 3-lane vertical scrolling gameplay
- Touch swipe controls (primary), keyboard/mouse (secondary)
- Procedurally generated obstacles and collectibles
- 4 distinct biomes (Forest, Mountain, Canyon, Rapids)
- Multiple game modes (Classic, Time Trial, Zen, Daily Challenge)
- Achievement system (50+ achievements)
- Power-ups (Shield, Magnet, Slow-Mo, Ghost, Multiplier)
- Local save/load (no backend, privacy-first)

## Design Principles

1. **Mobile-First**: Touch gestures, safe areas, orientation lock, haptics
2. **Ethical Design**: No dark patterns, no monetization, no tracking
3. **Accessibility**: WCAG 2.1 AA compliant, multiple input methods
4. **Performance**: 60 FPS target, < 2s load time
5. **Privacy**: Local-only data, no analytics, no user accounts

## Target Audiences

- Commuters (quick 2-5 minute sessions)
- Achievement hunters (completionists)
- Relaxation seekers (Zen mode)
- Competitive players (speedrunning)
- Families (parent-child bonding)

## Out of Scope

- Multiplayer (real-time or async)
- Global leaderboards (requires backend)
- User accounts / social login
- Monetization (ads, IAP, premium)
- Analytics / tracking
- Native mobile apps (PWA only for web)
